# Acknowledgements

MikuInvidious relies on the collective wisdom and reverse-engineering efforts of the open-source community. We would like to express our deepest gratitude to the following projects and individuals for their invaluable contributions to understanding the Bilibili API.

## Core Research & Documentation

- **[SocialSisterYi/bilibili-API-collect](https://github.com/SocialSisterYi/bilibili-API-collect)**: The foundational resource for Bilibili API documentation. Although the original repository has been archived, its legacy continues to power almost all third-party Bilibili tools.
- **[warterbili/-bilibili-](https://github.com/warterbili/-bilibili-)**: For providing comprehensive reverse-engineering records, Frida scripts, and deep insights into the `x-bili-ticket` mechanism, gRPC protocols, and App-specific signing algorithms.

## Libraries & Tools

- **[bilibili-api-python](https://github.com/nemo2011/bilibili-api)**: For providing a robust Python wrapper that simplifies interacting with Bilibili's complex API ecosystem.
- **The Frida Community**: For creating the essential instrumentation tools that made this research possible.
- **The Video.js & Media-Chrome teams**: For the excellent web player primitives.
